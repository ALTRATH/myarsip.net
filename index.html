  • String Config: vless://uuid@host:443?type=ws&security=tls&sni=[sni]&path=/vless#ArsipVLESS

  • Setup Dasar: 1. Buat akun VLESS di jagoanssh.com. 2. Di app: + > Import > Paste string > Edit SNI. 3. Path: /vless (default WS).
  • Connect: "VLESS Active". Speed 10-20Mbps, renew 7 hari.

Trojan
  • Elemen Penting: Password (plain text), SNI wajib (mimic HTTPS), Port 443. Gak ada payload kompleks.
  • String Config: trojan://pass@host:443?sni=[sni]&allowInsecure=0#ArsipTrojan
  • App Rekom: V2RayNG atau NapsternetV.
  • Setup Dasar: 1. Generate di sshocean.com. 2. Import > Edit SNI. 3. Fake Host: Tambah buat bypass.
  • Connect: No error = unlimited. Stabil game, low latency.

Shadowsocks (SS)
  • Elemen Penting: Password + Method (aes-256-gcm), Server/Port. Payload mirip SSH.
  • String Config: ss://method:pass@host:8388#ArsipSS
  • App Rekom: HTTP Injector (built-in) atau Shadowsocks app.
  • Setup Dasar: 1. Buat server SS di jagoanssh.com. 2. Shadowsocks tab > Masukin host: id.ss.jagoanssh.com, port: 8388, pass/method. 3. Payload (ops): CONNECT [host_port] HTTP/1.1[crlf]Host: [host][crlf][crlf].
  • Start: "SS Tunnel Up". Cepet daily, method ganti chacha20 kalau lambat.
  • Update Nov 2025: Sederhana buat basic inject.
